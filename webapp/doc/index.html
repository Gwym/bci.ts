Bci.ts

    English

    Presentation

    A node.js Web server to control OpenBCI 32 v3 board (http://www.openbci.com/) from browser and store data to file.

    Get started

    Install dependencies and launch server :

    npm --prefix ./build/ install
node ./build/server/app.js

    Point your browser to server address set in configuration (default : http://127.0.0.1:8080/ ).

    (Tested with NodeJs v5.0.0 and Chromium Version 47 on Ubuntu 15.10 64)

    Français

    Présentation

    Un serveur Web Node.js pour contrôler la carte OpenBCI 32 v3 (http://www.openbci.com/) depuis le navigateur et pour enregistrer les données sur disque.

    Pour débuter

    Installer les dépendences et lancer le serveur :

    npm --prefix ./build/ install
node ./build/server/app.js

    Dans votre navigateur, allez à l'adresse définie dans la configuration (par défault : http://127.0.0.1:8080/ ).

    (Testé avec NodeJs v5.0.0 et Chromium Version 47 sous Ubuntu 15.10 64)
